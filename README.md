# rusdy.com

Personal website for RA (Rusdy Ahmad), built with Deno + Fresh.

## Requirements

- Deno (https://docs.deno.com/runtime/getting_started/installation)

## Development

```sh
# start dev server

deno task dev
```

## Formatting & Linting

```sh
# format

deno task format

# lint

deno task lint
```

## Type Check

```sh

deno task typecheck
```

## Build

```sh

deno task build
```
